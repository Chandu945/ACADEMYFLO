## Generated Artifacts

| Artifact | Path | Generated By |
|----------|------|--------------|
| Traceability JSON | `artifacts/handoff/traceability.json` | `npm run handoff:generate` |

## Generating Artifacts

```bash
npm run handoff:generate
```

This runs both `scripts/handoff/generate-traceability.mjs` and `scripts/handoff/generate-handoff-summary.mjs`.
